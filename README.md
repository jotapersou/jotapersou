## Sobre Mim

Atualmente curso Análise e Desenvolvimento de Sistemas e estou em processo de transição para a área de tecnologia.

Possuo experiência profissional em ambiente industrial, atuando com processos produtivos, organização operacional, resolução de problemas e trabalho em equipe. Essa trajetória contribuiu para o desenvolvimento de habilidades importantes como comunicação interpessoal, responsabilidade, disciplina e adaptação a diferentes cenários.

Atualmente estudo desenvolvimento web, lógica de programação, versionamento de código e análise de sistemas, buscando aprimorar continuamente minhas habilidades através de projetos práticos e estudos complementares.
